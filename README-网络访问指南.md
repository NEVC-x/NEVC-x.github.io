# 随文识字应用 - 网络访问指南

## 🌐 如何让任何人都能访问

应用现在已经配置为可以在同一网络下被任何人访问！

### 访问地址

- **本地访问**: http://localhost:3001
- **网络访问**: http://192.168.1.15:3001

### 使用步骤

1. **确保服务器正在运行**
   - 服务器已经在后台运行（端口 3001）
   - 如果停止了，可以运行 `npm run start:server` 重新启动

2. **获取正确的IP地址**
   - 如果上面的IP地址不正确，可以打开命令提示符
   - 运行 `ipconfig` 查看 "IPv4 地址"
   - 使用实际的IP地址替代 `192.168.1.15`

3. **访问方式**
   - 在同一网络下的任何设备（手机、平板、电脑）上打开浏览器
   - 输入 `http://[你的IP地址]:3001`
   - 例如：`http://192.168.1.10:3001`

### 示例

假设你的电脑IP地址是 `192.168.1.10`，那么：
- 手机访问：`http://192.168.1.10:3001`
- 平板访问：`http://192.168.1.10:3001`
- 另一台电脑：`http://192.168.1.10:3001`

### 故障排除

1. **无法访问？**
   - 检查防火墙是否阻止了端口 3001
   - 确保所有设备在同一个WiFi/网络下

2. **如何查找IP地址？**
   ```
   在Windows上：
   1. 按 Win + R，输入 cmd
   2. 输入 ipconfig
   3. 找到 "IPv4 地址"

   在Mac上：
   1. 打开系统偏好设置 > 网络
   2. 选择当前连接的网络
   3. 查看 "IP地址"
   ```

3. **需要更改端口？**
   - 编辑 `server.js` 文件
   - 修改 `const PORT = process.env.PORT || 3001;` 中的端口号

### 其他设备访问示例

| 设备类型 | 访问方式 |
|---------|---------|
| 手机/平板 | 打开浏览器，输入IP地址 |
| 其他电脑 | 输入 `http://[IP]:3001` |
| 智能电视 | 浏览器输入IP地址 |

### 停止服务器

要停止服务器，请按 `Ctrl+C` 关闭命令行窗口。

### 注意事项

- 默认只在同一网络内可见
- 如果需要互联网访问，需要配置端口转发（需要路由器管理权限）
- 建议用于教育或演示场景
