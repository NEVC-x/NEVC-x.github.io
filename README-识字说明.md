# 随文识字应用使用说明

## 功能介绍

这是一个基于 React 开发的随文识字学习应用，主要功能包括：

1. **课文输入**：可以输入任意中文课文，系统会自动识别生字并高亮显示
2. **生字查询**：点击课文中的蓝色生字，查看详细的拼音、释义和笔顺
3. **快速搜索**：在搜索框输入任意汉字，快速查询字典中的汉字信息
4. **学习统计**：自动统计课文的字数、生字数量和生字占比

## 使用方法

### 1. 输入课文
- 在左侧文本框中输入要学习的中文课文
- 系统会自动识别出字典中的生字并用蓝色高亮显示

### 2. 查看生字信息
- 点击课文中的蓝色生字，会弹出详细信息卡片
- 卡片中包含：
  - 汉字显示
  - 拼音
  - 释义
  - 笔顺动画
  - 相关例词
- 点击卡片可以听到汉字发音
- 点击背景或关闭按钮关闭卡片

### 3. 快速搜索汉字
- 在搜索框中输入想要查询的汉字
- 点击"查询"按钮查看该字的详细信息
- 点击"×"按钮清除搜索

### 4. 生字表
- 右侧显示当前课文中的所有生字
- 点击生字表中的字也可以查看详细信息

### 5. 学习统计
- 右下角显示课文的学习统计数据
- 包括总字数、生字数量和生字占比

## 技术特点

1. **响应式设计**：适配不同屏幕尺寸，在手机和电脑上都能正常使用
2. **动画效果**：使用 hanzi-writer 库实现真实的汉字笔顺动画
3. **发音功能**：使用 Web Speech API 实现汉字发音
4. **用户友好**：界面简洁直观，操作简单

## 支持的汉字

目前内置了30多个常用汉字，包括：
- 基础字：我、你、他、大、小、天、地、水、火、木、人、口
- 课文字：学、京、剧、唱、戏、跟、老、师、很、好
- 其他常用字：山、水、火、木、人、口、天、地等

## 扩展功能

如需添加更多汉字，可以在 TextReader.js 文件中的 EXTENDED_DICTIONARY 对象中添加新字的数据格式：

```javascript
'字': {
  pinyin: "zì",
  meaning: "含义解释",
  strokes: [
    "M10 10 L90 10",
    // ... 笔画路径数据
  ],
  examples: ["例词1", "例词2", "例词3"]
}
```

## 运行项目

要启动项目，请在项目目录下运行：
```bash
npm start
```

项目会在浏览器中自动打开，地址通常是：http://localhost:3000
