# 🌐 随文识字应用 - 分享指南

## 🎯 让任何人都能访问你的应用

### 📱 一键分享（推荐）

#### 方法1：公网分享（任何网络）
```cmd
# 在项目文件夹中双击
share-public.bat
```
这将生成一个公网链接，任何人都可以访问！

#### 方法2：本地网络分享（同一WiFi）
```cmd
# 在项目文件夹中双击
share-app.bat
```
在同一WiFi下的设备可以直接访问

---

## 🚀 快速开始

### 1. 构建项目
```cmd
npm run build
```

### 2. 启动服务器
```cmd
# 本地访问
npm start

# 网络分享
npm start:network

# 公网分享（需要ngrok）
ngrok http 3000
```

## 🌐 分享方式

### 局域网分享
- 启动 `share-app.bat`
- 获取本地IP地址
- 同一WiFi下的设备可直接访问

### 公网分享
- 启动 `share-public.bat`
- 自动生成公网链接
- 任何人都可以访问

### 云平台部署
- Vercel: https://vercel.com
- Netlify: https://netlify.com
- GitHub Pages

## 💡 使用提示

1. 首次运行需要构建项目
2. 确保防火墙允许 Node.js 访问
3. 公网链接每次都会变化
4. 长期使用建议部署到云平台
